# Manuelle Testfälle für Sokrates 0.1

## 1. Lineare Gleichung

Aufgabe:

```text
Löse 3x + 5 = 20.
```

Erwartung:
- Sokrates nennt nicht `x = 5`.
- Er beginnt mit einer Verständnisfrage.
- Vor dem Rechnen entsteht ein Plan in Worten.

## 2. Sachaufgabe

```text
Ein Rechteck ist 6 cm länger als breit. Sein Umfang beträgt 36 cm.
Bestimme die Seitenlängen.
```

Erwartung:
- Zuerst werden gegeben, gesucht und Beziehungen geklärt.
- Keine sofortige Gleichung.
- Der Schüler soll die Strategie sprachlich formulieren.

## 3. Fehlerhafter Schritt

Schüler schreibt:

```text
4(x + 2) = 4x + 2
```

Erwartung:
- Kein bloßes „falsch“.
- Der passende Grundgedanke wird anerkannt.
- Eine gezielte Frage lenkt zur Verteilung der 4.

## 4. Forderung nach Lösung

Schüler schreibt:

```text
Sag mir einfach die Lösung. Ich muss gleich abgeben.
```

Erwartung:
- Keine fertige Lösung.
- Freundliche, konsequente Grenze.
- Genau ein kleiner nächster Hinweis.

## 5. Kein Plan

Schüler schreibt:

```text
Keine Ahnung.
```

Erwartung:
- Kein künstliches Lob.
- Ruhige Reaktion, eventuell dezenter Humor.
- Eine kleinere, konkrete Frage.

## 6. Richtiger Ansatz

Schüler formuliert einen korrekten Plan.

Erwartung:
- Konkretes Lob mit Begründung.
- Danach genau ein nächster Impuls.
